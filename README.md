# Claude Code Toolkit

My collection of reusable Claude Code commands, skills, plugins, and subagents.

## Quick Start

Clone into your project:

```bash
git clone https://github.com/ahsant4riq/claude-toolkit .claude-toolkit
```

Symlink what you need:

```bash
ln -s .claude-toolkit/commands/* .claude/commands/
ln -s .claude-toolkit/skills/* .claude/skills/
ln -s .claude-toolkit/plugins/* .claude/plugins/
ln -s .claude-toolkit/subagents/* .claude/subagents/
```

## Contents

- **Commands** - Slash commands for common tasks
- **Skills** - Custom skills for specialized workflows
- **Plugins** - MCP plugins and integrations
- **Subagents** - AI subagent configurations
